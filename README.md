## Быстрый запуск
### Makefile
Доступные команды:
```bash
make start - запуск программы
make stop - остановка программы
make logs - отображение логов
make reset - очистка и перезапуск программы
```

### Windows
```bash
run.bat - CLI меню для быстрого управления
stop.bat - принудительная остановка программы
```

### Linux
```bash
run.sh - CLI меню для быстрого управления
stop.sh - принудительная остановка программы
```
